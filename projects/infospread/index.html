Motivation

In the digital age, understanding and controlling how misinformation spreads across networks is paramount. Controlling how opinions propagate through large social networks is inherently a planning problem — given a network state, which nodes should receive an intervention (accurate information), and in what sequence, mitigate harmful beliefs and reduce the infection rate — the percentage of the network currently holding misinformation.

As interactions continuously shape opinions, traditional approaches that model belief as static or binary lack realism. We propose frameworks that integrate generalized automated planning with opinion dynamics modeling, treating intervention as a sequential decision-making process constrained by an execution budget.

Modeling Dynamic Opinion Networks

We model opinion networks as directed graphs $G = (V, E)$ where each node holds a continuous opinion $x_i(t) \in [-1, 1]$ and each edge carries a trust weight $\mu_{ik} \in [0, 1]$. An agent is considered "infected" if their opinion value falls below a specific threshold (e.g., $x_i < -0.95$).

Infection Rate: A primary metric for network health, defined as the ratio of infected nodes to the total number of agents in the network:

$\text{Infection Rate}(t) = \frac{|\{v \in V : x_v(t) < -0.95\}|}{|V|}$

We study cases ranging from simple binary models to fully continuous opinions and trust using linear adjustment mechanisms.

Linear Adjustment Mechanism: Opinions evolve over time as individuals communicate. When agent $k$ shares information with agent $i$, the belief updates via:

$x_i(t+1) = x_i(t) + \mu_{ik}(x_k(t) - x_i(t))$

The goal of the planner is to intelligently select a budget-constrained subset of nodes at each step to inject with accurate information, countering misinformation spread.

Intervention planning strategies for dynamic opinion networks

Combinatorial node selection on graphs quickly makes optimal search planning intractable. We leverage the structural properties of opinion networks to develop scalable, generalized strategies that transfer across different network topologies.

Supervised Learning (SL) with GCNs

The motivation for applying SL with Graph Convolutional Networks (GCNs) stems from the "generalized planning" challenge: how can a system learn to select nodes on a small 10-node network and have that strategy hold for a 10,000-node network? GCNs excel here by learning structural features (connectivity, local polarity, influence) rather than memorizing node IDs.

We formulate node selection as a ranking problem, using an optimal search algorithm to generate ground-truth labels for small topologies. The GCN learns to replicate these optimal choices by processing the network as a graph convolution layers, capturing the cascading effects of opinion spread.

Reinforcement Learning (RL) & Reward Engineering

While SL is effective, it suffers from the labeling bottleneck: generating optimal labels for large-scale graphs requires solving NP-hard search problems. To eliminate this dependency, we introduced a Reinforcement Learning framework powered by a Deep Value Network (DVN).

The RL Advantage: Our framework eliminates the need for expensive supervised labels by learning directly from simulator feedback. This approach discovers robust strategies that generalize across diverse network topologies and scales.

A core contribution of our research (presented at NeurIPS 2024) is the analysis of 6 distinct reward structures. These formations dictate how the agent balances global network health with local observable metrics:

Global Absolute
$R_0 = \sum_{v \in V} I(o_v = o_{tar})$
Global Density
$R_1 = \frac{1}{|V|} \sum x_v$
Step-wise Delta
$R_2 = R_1(t) - R_1(t-1)$
Local Neighborhood
$R_3 = \sum_{v \in \mathcal{N}(L)} x_v$
Influence-Weighted
$R_4 = \sum \text{deg}(v) \cdot x_v$
Closeness Payoff
$R_5 = \sum \gamma^{d(v, L)} \cdot x_v$

InfoSpread: System Implementation

To transition our theoretical frameworks into practical tooling, we developed InfoSpread, a system bridging abstract planning formalisms with interactive visualization workflows.

Numeric PDDL Formulation: We provide the first numeric PDDL representation strictly tailored to capture opinion dynamics. It utilizes conceptual types (\texttt{agent}, \texttt{source}, \texttt{topic}) and relies on fluents like \texttt{have-opinion} and \texttt{have-trust} alongside a foundational \texttt{spread-info-from} action to systematically drive propagation mathematically on off-the-shelf planners like Metric-FF.

The platform translates configurations into PDDL problems, triggers back-end planners, and visualizes stepping plans. This fosters collaborative human-AI dynamics—providing strategists with visual feedback and manual override capabilities during sequence interventions against cyber threats.

Extension to Dialog Management

Recent work leverages these opinion propagation rules to drive Conversational Dialog Management. Rather than isolated nodes, we consider a user’s interrelated beliefs across connected topics. As chat agents emit utterances, persuasion alters an interconnected belief state modeled as an induced opinion network, creating intelligent guides towards holistic behavioral changes (e.g., civic participation or safe social media activity).
